# XPL0R3R

**Code:** EXP  
**Remote:** https://github.com/Lexxicc/RPGX.git  
**Version:** v0.0.5 (`XPL0R3R v0.0.5.html`)  
**Genre:** RPG / exploration

## Tech
- Canvas-based rendering (4 canvas elements — most complex canvas setup in the studio)
- Vanilla JS + Tailwind CSS via CDN
- Single-file HTML — no build step

## localStorage
- `exp_achievements` — unlocked achievements
- `exp_boss1_cleared` — boss clear flag
- `exp_character` — saved character state
- `exp_event_log` — in-game event history
- `exp_highscores` — high scores
- `exp_last_journey` — last run state
- `exp_leftHanded` — left-hand control preference

## Notes
- GitHub repo is named `RPGX` (original working title)
- Most localStorage keys of any game in the studio
- Left-handed mode toggle (same as XDS.Dungeon-Swingers)
